# Goolride Amazon Deals

Static link-tree page for daily Amazon tech deal links.

## Local Preview

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Cloudflare Pages project: `goolride-amazon-deals`

Primary domain target: `Goolride.com`

## Affiliate Tag

If you need to add an Amazon Associates tag, edit `affiliateTag` near the bottom of `public/index.html`.
